##### 質問・疑問・感想・シェアしたいこと等
感想：プログラミング、サイトデザイン、そしてビジネスのスキルそしてセンスは全く別々のものであることを痛いほど理解した。技術的にプログラミングを理解していても思い描いている美しいサイトはつくれず、どれだけきれいなサイトをつくれてもビジネスの中身がないと意味がない。まずは自ら全てのアスペクトを経験し、できる・やりたいところは自分でやり、できない・やりたくないところは外だしする（長期的なはなし）。
